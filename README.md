# Smart City Backend

## Ngrok Test Environment

This branch is dedicated to running an Ngrok server for testing external triggers and third-party callbacks to our app, since localhost cannot be accessed externally.

* This branch should **only** be run from the Codespace **`ngrok-tunnel-server`**.
* This branch will **not** be merged into `main`.
* It exists solely for testing purposes until a valid production domain is available.
